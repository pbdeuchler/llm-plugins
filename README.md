# llm-plugins

Largely stolen from [ed3d-plugins](https://github.com/ed3dai/ed3d-plugins) and ever so slightly modified

### house-style

Development guides per my tastes

## Installation

### Add the marketplace

```bash
/plugin marketplace add https://github.com/pbdeuchler/llm-plugins.git
```

### Install plugins

All plugins are available from the `llm-plugins` marketplace:

```bash
/plugin install house-style@llm-plugins
# ... etc
```
